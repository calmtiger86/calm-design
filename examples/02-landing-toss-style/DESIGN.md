# Design System: AI 영상 편집 SaaS 랜딩페이지 (토스 스타일 영감)

> Generated by calm-design · LANGUAGE: ko · VARIANCE: 7 · MOTION: 6 · DENSITY: 4 · Mode: A
> Inspired by: Toss (toss.im) — 한국 핀테크 랜딩의 정제된 톤·여백 활용

## 1. Visual Theme & Atmosphere

토스 핀테크 톤의 차분한 정제와 SaaS 트렌디함의 균형. 충분한 화이트스페이스, 단일 Toss Blue 액센트, 큰 Pretendard 헤드라인, 부드러운 Spring 모션. "이거 잘 만들었네"라는 인상보다 "쓰기 편할 것 같다"는 신뢰가 우선. 인라인 image typography 1회 사용으로 차별화.

## 2. Color Palette & Roles

| 이름 | Hex | 역할 |
|---|---|---|
| Canvas | `#FFFFFF` | 페이지 배경 (순백) |
| Surface Soft | `#F9FAFB` | 카드·섹션 배경 (Zinc-50) |
| Ink | `#191F28` | 본문·헤드라인 (Toss 톤 유사 다크) |
| Mute | `#8B95A1` | 보조 텍스트 (Toss 톤 그레이) |
| Border | `#E5E8EB` | 분할선 |
| Accent | `#3182F6` | 단일 액센트 — Toss Blue (CTA, 링크, 강조) |
| Accent Soft | `#E8F3FF` | 액센트 배경 (배지·강조 박스) |

**필수 규칙**: Toss Blue 1개 액센트만, Pure Black 절대 X (Ink는 `#191F28`), 채도 75% 미만.

## 3. Typography Rules

**한국어 (강제)**:
- Family: `Pretendard Variable, Pretendard, system-ui, sans-serif`
- Hero H1: `text-5xl md:text-6xl lg:text-7xl font-bold tracking-tight leading-tight word-break:keep-all`
- Section H2: `text-3xl md:text-5xl font-bold tracking-tight word-break:keep-all`
- 본문: `text-base md:text-lg leading-relaxed text-mute max-w-[65ch]`
- CTA 라벨: `text-base font-semibold`

**금지**: Inter, Noto Sans KR, font-thin/extralight/light.

## 4. Component Stylings

**Button (Primary CTA)**
- Default: `bg-[#3182F6] text-white rounded-full px-7 py-3.5 font-semibold text-base`
- Hover: `bg-[#1B64DA] scale-[1.02]` 300ms cubic-bezier
- Focus: `ring-2 ring-[#3182F6] ring-offset-2`
- Active: `scale-[0.98]`
- Disabled: `opacity-50`
- Loading: 인라인 spinner

**Button (Secondary)**
- Default: `bg-zinc-50 text-ink rounded-full px-7 py-3.5 font-semibold border border-zinc-200`
- Hover: `bg-zinc-100`

**Feature Card (Bento 큰 카드)**
- `bg-white rounded-3xl border border-zinc-200 p-8 md:p-10`
- 큰 아이콘 또는 일러스트 상단, 헤드라인 중앙, 설명 하단

**Feature Card (Bento 작은 카드)**
- `bg-white rounded-2xl border border-zinc-200 p-6`

## 5. Layout Principles

- **Section Padding**: `py-24 md:py-32 lg:py-40` (대형 호흡)
- **Max-width**: `max-w-7xl mx-auto px-4 md:px-6`
- **Hero**: Split 60/40 (텍스트 좌, 비주얼 우) 또는 Full-bleed asymmetric
- **Features**: Bento Grid — 큰 카드 1 (`col-span-2 row-span-2`) + 작은 카드 4
- **Testimonials**: Masonry (3-column 비대칭, 다른 카드 높이)
- **VARIANCE=7 강제**: 인접 섹션 다른 레이아웃
  - Hero (Split) → Social Proof (Logo Cloud) → Features (Bento) → Use Cases (Zig-Zag) → Testimonials (Masonry) → Pricing (Comparison) → CTA (Full-bleed) → Footer (Minimal)

- **금지**: 3-column equal cards, centered hero (VARIANCE > 4), `h-screen`

## 6. Depth & Elevation

- **그림자**: 프리미엄 카드는 `shadow-[0_4px_20px_rgba(25,31,40,0.04)]` (Toss 톤 매우 부드러움)
- **Sticky Nav**: `bg-white/80 backdrop-blur-md` + 하단 1px 보더
- **Hero 비주얼**: `rounded-3xl shadow-[0_8px_40px_rgba(49,130,246,0.08)]` (Accent 색 미세 틴팅)
- **금지**: 네온 글로우, Pure Black 그림자

## 7. Motion & Interaction

- **Spring 기본**: 600ms `cubic-bezier(0.16, 1, 0.3, 1)` — Toss의 부드러운 톤
- **Hero 진입**: 텍스트 stagger fade+up 100ms, 비주얼은 200ms 지연
- **CTA hover**: scale + 색상 전환 300ms
- **숫자 카운트업** (Social Proof): `IntersectionObserver` 기반, 1초
- **Feature Card hover**: 미세 `translateY(-2px)` + 보더 강조

## 8. Responsive Behavior

- **Breakpoints**: 모바일 `<768px`, 태블릿 `768-1024px`, 데스크톱 `≥1024px`
- **Hero Split**: 모바일 1-column (텍스트 위, 비주얼 아래)
- **Bento Grid**: 모바일 1-column, 태블릿 2-column, 데스크톱 비대칭 3-column
- **Touch targets**: ≥44x44px
- **Nav**: 모바일 hamburger drawer

## 9. Anti-Patterns (Project-Specific Banned)

- ❌ 광고적 카피: "혁신적인 AI 영상 편집", "차원이 다른" — Toss 톤은 사실·기능 강조
- ❌ 가짜 통계: "10만 사용자", "99.9% 만족도" 같은 검증 안 된 수치
- ❌ Stock 이미지 사용자 얼굴: 진짜 사용자 또는 추상적 일러스트
- ❌ 채팅 봇 floating button (랜딩 단계에서는 방해)
- ❌ 자동재생 비디오 음성 ON
